how to use this app

hello! thank you so much for choosing to use dodue.

this app was created as a minimal, simple, and quick way to manage due dates, but also help plan when you actually want to do a task.

creating tasks & editing task information

to create a task, fill out all of the fields and then press the "enter" key in any of the new task input fields.

to edit a name or do/due date, just click on the text in the task box, and it will be replaced with an editable input element.

while editing, press the "enter" key to save your changes, or press the "escape" key if you want to exit without saving changes.

selecting a task

to select a task, you can either click the task, use the "1" through "9" keys to select a specific task, or use the "up" and "down" keys. you can also move the task cursor with these keys.

you can exit selection mode at any time by pressing the "escape" key.

while a task is selected, press "d" to delete a task (it'll ask you to confirm to prevent any accidents). you can also press the "q", "w", or "e" keys to edit the name, do date, or due date, respectively.

miscellaneous

if you wish to quickly jump to the new task creation area, you can use the "n" key to focus on the new task name input.

you can show or hide a task section simply by clicking on the header. the header will display the number of tasks in that section while the section is hidden.
