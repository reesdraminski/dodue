how to use this app

hello! thank you so much for choosing to use dodue.

to edit a name or do/due date, just click on the text in the task box, and it will be replaced with and editable element.

while editing, press the "enter" key to save your changes, or press the "escape" key if you want to exit without saving changes.

pro tip: you can press the "enter" key in any of the new task input fields to create a new task instead of having to press the "add task" button.

to enter keyboard navigation mode, press the "tab" key (but not while you're editing any inputs or anything like that). once you're in keyboard navigation mode, you can exit by pressing the "escape" key.

you can move the task cursor by pressing the "up" or "down" arrow keys. press "d" to delete a task (it'll ask you to confirm to prevent any accidents).

if you have a task highlighted during the keyboard navigation mode, you can press the "1", "2", or "3" keys to edit the name, do date, or due date, respectively.

if you are not in keyboard navigation mode, you can use the "n" key to focus on the new task name input.
